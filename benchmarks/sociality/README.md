# sociality benchmarks

Deterministic JSON fixtures for the six initial sociality scenarios:

1. focused work + low energy
2. direct human question awaiting reply
3. two similar mugs with reference resolution
4. late-night posting review
5. commitment persistence across restart
6. relationship continuity based on same-day disclosure

Run the small harness with:

```bash
python benchmarks/sociality/run_suite.py benchmarks/sociality/fixtures
```
